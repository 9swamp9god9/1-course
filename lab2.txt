Задание 1
1.1
Пользователь вводит своё имя, которое сохраняется в переменную name.
Функция greet принимает имя и печатает приветственное сообщение.
Вызывается функция greet, которая выводит, например, "Привет, Роман".
1.2
Пользователь вводит число, сохраняемое в переменной number.
Функция square принимает это число и печатает его квадрат.
Вызывается функция square, которая вычисляет и выводит квадрат введенного числа.
1.3
Пользователь вводит два числа x и y.
Функция max_of_two сравнивает эти числа и печатает большее из них, или сообщение "Числа равны", если они одинаковые.
Вызывается функция max_of_two, которая выполняет это сравнение.

Задание 2
2.1
Пользователь вводит число number.
Функция is_prime проверяет, является ли число простым:
Используется цикл, который делит число на все числа от 2 до number - 1.
Если число делится без остатка на любое из них, функция возвращает False.
Если оно не делится ни на одно из этих чисел, возвращает True.
Вызывается функция is_prime, которая выводит True для простого числа и False для составного.
2.2
Пользователь вводит имя и возраст.
Функция describe_person принимает имя и возраст, возвращает строку с описанием человека.
Если возраст не задан, используется значение по умолчанию 30.
Вызывается функция describe_person, которая возвращает строку, например, "Меня зовут Роман, мне 18 лет", и эта строка выводится на экран.
