1 задание

Создаем файл example.txt
Функция read_file открывает файл example.txt
Аргумент type определяет, как обрабатывается содержимое файла:
Если type равен 1, читается и возвращается весь файл как строка.
Если type равен 2, читается файл построчно, и каждая строка добавляется в список lst без символов новой строки.
Возвращается строка под номером num_line (учитывая, что индексация начинается с 1).
Если type не равен ни 1, ни 2, возвращается сообщение об ошибке.
Пример вызова read_file(2, 1) возвращает первую строку файла.

2 задание

Функция new_file открывает файл user_input.txt в режиме добавления (append) и чтения (a+), создавая файл, если он не существует.
Текст из аргумента text записывается в файл.
После записи выводится сообщение "Данные записаны".
В примере new_file('AaaaAAAAaaaa') функция добавляет строку 'AaaaAAAAaaaa' в файл.

3 задание

Функция read_file принимает три аргумента: filename, type, и num_line.
Открывается файл с именем filename. Если файл не существует, обрабатывается исключение FileNotFoundError, и возвращается сообщение об ошибке.
Обработка файла аналогична заданию 1:
Если type равен 1, возвращается весь контент.
Если type равен 2, возвращается строка с номером num_line из файла.
Если type не соответствует ни 1, ни 2, возвращается сообщение об ошибке.
В примере read_file('/Users/swampgod/Desktop/example.txt', 2, 4) возвращается четвертая строка из файла example.txt, если она существует.
